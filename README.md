# market-ml

## SETUP

### Required software
```bash
apt install python3 pip pipenv
```

### Install dependencies
```bash
pipenv install -e .
pip install -e .
```

### Execution (direct)
```bash
pipenv run python3 scripts/main.py 
```

### Execution
```bash
pipenv shell
python3 python3 scripts/main.py 
```
