# market-ml

## SETUP

### Required software
```bash
apt install pip pipenv
```

### Install dependencies
```bash
pip install -e .
```
OR
```bash
pip install .
```

### Execution (direct)
```bash
pipenv run python marketml/Regressor/Regressor_fun.py 
```

### Execution
```bash
pipenv shell
python3 marketml/Regressor/Regressor_fun.py 
```
